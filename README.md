# Netflix-Data-Visualization-Dashboard
# 📊 Netflix Data Visualization using Matplotlib

## 📌 Project Overview
This project analyzes the Netflix dataset and visualizes key insights using **Python and Matplotlib**. The goal is to understand the distribution of Netflix content such as **Movies vs TV Shows, content ratings, and movie duration patterns**.

The project demonstrates how **data cleaning, filtering, and visualization** can be used to extract meaningful insights from real-world datasets.

---

## 📂 Dataset
The dataset used in this project is:

**netflix_titles.csv**

It contains information about Netflix content including:

- Show ID
- Type (Movie / TV Show)
- Title
- Director
- Cast
- Country
- Date Added
- Release Year
- Rating
- Duration
- Genre (Listed In)
- Description

---

## 🛠 Technologies Used

- Python
- Jupyter Notebook
- Pandas
- Matplotlib

---

## 📊 Analysis and Visualizations

The following visualizations were created in this project:

### 1️⃣ Movies vs TV Shows Distribution
A **bar chart** showing the number of Movies compared to TV Shows available on Netflix.

### 2️⃣ Content Rating Distribution
A **pie chart** visualizing the percentage of different content ratings (TV-MA, PG, R, etc.).

### 3️⃣ Movie Duration Distribution
A **histogram** showing the distribution of movie durations to understand common movie lengths on Netflix.

---

## 📈 Key Insights

- Netflix contains a **larger number of Movies compared to TV Shows**.
- Content ratings vary widely, with some ratings appearing much more frequently.
- Most movies fall within a **common duration range**, showing typical streaming movie lengths.
